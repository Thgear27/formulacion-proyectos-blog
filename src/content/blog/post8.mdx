# Estructura de Desglose del Trabajo (EDT)

La Estructura de Desglose del Trabajo (EDT), es una herramienta que organiza un proyecto dividiéndolo en partes más pequeñas y fáciles de manejar. Es una lista de tareas donde cada una representa un paso específico para lograr el objetivo final. Esta organización permite saber exactamente qué hace. Esto permite que el proyecto se vuelva más claro y fácil de controlar en cada etapa.
