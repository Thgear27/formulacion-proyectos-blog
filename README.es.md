1. Clona o haz un [fork](https://github.com/danielcgilibert/blog-template/fork) del repositorio:

2. Instala las dependencias:

```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Ejecuta el servidor de desarrollo:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
